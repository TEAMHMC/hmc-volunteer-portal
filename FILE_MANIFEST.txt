KEY v4.0 IMPROVEMENTS
======================

1. TYPES.TS - New fields in Volunteer interface:
   - identityLabel: 'HMC Champion'
   - volunteerRole: specific role (separate from identity)
   - coreVolunteerStatus: boolean (training milestone)
   - completedHIPAATraining: boolean (mandatory)
   - eventEligibility: { gates for event types }
   - availability: { days, time, startDate, etc }
   - compliance.hipaaTraining: ComplianceStep
   - createdAt, updatedAt, lastActiveAt

2. CONFIG.TS - Updated role structure:
   - PRIMARY: Core Volunteer (community pathway)
   - SPECIALIZED: 18 parallel role tracks
   - Each role has own compliance, training, eligibility
   - BUSINESS_RULES object for CSV import & validation

3. SHIFT_REGISTRATION_VALIDATION.ts - NEW SERVICE:
   - validateShiftRegistration() function
   - 5 validation gates (Core status, HIPAA, training, BG check, event gate)
   - Clear error messages for users
   - Helper functions for promotion and status checks

4. CSVImportService.ts - NEW SERVICE:
   - parseCSV() - Parse CSV files
   - validateRow() - Validate data
   - rowToVolunteer() - Convert to Volunteer objects
   - importCSV() - Full import with error reporting
   - generateTemplate() - CSV template for users
   - Supports up to 1000 volunteers per import
